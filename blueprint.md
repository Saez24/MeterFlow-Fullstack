# Blueprint

## Overview

MeterFlow is a modern web application for managing energy consumption data. It allows users to track meter readings, manage tariffs, and visualize their consumption data.

## Features Log

*   **Meter Management**: Create, edit, and delete meters for different energy types (electricity, gas, water).
*   **Readings Management**: Record meter readings, including optional photos. The app automatically calculates consumption between readings.
*   **Tariff Management**: Define tariffs with base charges and price per unit. The app can handle multiple tariffs over time.
*   **Data Visualization**: View consumption data in charts.
*   **Cost Preview**: A live cost preview to estimate annual costs, refunds, or additional payments based on estimated consumption and tariff data.

## Current Plan

The application is in a good state. Future features could include:

*   More advanced reporting and data export.
*   User authentication and multi-user support.
*   Reminders for meter readings.
